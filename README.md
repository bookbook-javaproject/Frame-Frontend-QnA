# Frame-Frontend-QnA

Frame Q&amp;A system front

## 사용방법

```bash
yarn add https://github.com/bookbook-javaproject/Frame-Frontend-QnA.git
```

## 사용 예시

```js
<template>
  <messenger />
</template>

<script>
import Messenger from 'frame-qna';

export default {
  name: 'App',
  components: { Messenger },
};
</script>
```
